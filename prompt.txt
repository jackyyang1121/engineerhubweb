# 專案背景
我要開發一個專屬於工程師的討論社群 Web App，目標是打造一個充滿科技消息、程式碼分享、技術討論與觀點交流的平臺。用戶可以發表文字、圖片、影片與程式碼片段（支援程式碼高亮），互動功能包括點讚、留言、轉發、私訊即時聊天。系統有搜尋功能，也要支援 Google 和 GitHub 登入，並用 JWT + OAuth2 做認證授權。UI 要簡約且響應式。

# 技術棧
- 後端：Python + Django REST Framework，提供 REST API  
- 資料庫：PostgreSQL  
- 即時聊天：Django Channels + WebSocket + Redis  
- 搜尋：Algolia  
- 認證授權：JWT + OAuth2 (Google, GitHub登入)  
- 前端：React + Vite + TypeScript + Tailwind CSS  
- 程式碼高亮顯示：Pygments（後端處理程式碼語法高亮）  

# 需求功能（重點摘要）
1. **使用者系統**
   - 註冊（手機號碼/電子信箱 + 密碼 + 用戶名稱）  
   - Google與GitHub OAuth登入  
   - JWT驗證與授權  

2. **首頁資訊流**
   - 顯示追蹤用戶貼文、熱門貼文  
   - 滑動瀏覽貼文  
   - 貼文內容支援文字、圖片、影片、多達10張的媒體輪播  
   - 貼文可包含100行以內程式碼區塊，點擊展開查看完整程式碼  
   - 程式碼高亮顯示並支援「一鍵複製」  

3. **貼文互動**
   - 點讚、留言、轉發、分享、儲存  
   - 貼文編輯（修改文字、媒體、程式碼）  
   - 貼文檢舉功能  

4. **即時聊天**
   - 私訊功能  
   - 顯示最近聊天對象、已讀/未讀狀態、可發送文字、表情符號與媒體檔  

5. **搜尋功能**
   - 搜尋用戶名稱（部分匹配）  
   - 搜尋貼文關鍵字（含程式碼片段關鍵字）  
   - 顯示近期搜尋紀錄並可清除  

6. **個人檔案頁**
   - 顯示用戶自介、技能標籤、頭像  
   - 追蹤數、被追蹤數  
   - 作品集展示（可附 GitHub、作品網址、YouTube）  
   - 查看別人個人頁時顯示追蹤與私訊按鈕  
   - 自己個人頁時可編輯個人資料
